# RUIN

Ruin is a simple ofx import with budget capabilities.

# Usage

```
Generates budget data from ofx files

Usage:
  ruin [command]

Available Commands:
  delete      Delete a budget
  help        Help about any command
  import      import an ofx file into the ledger
  list        List budgets or transactions
  modify      Modify a budget, a transaction or a category
  reindex     A brief description of your command
  report      Show reporting with budget difference
  show        Show a transaction

Flags:
      --file string   Ledger file (default "$HOME/.ruin.json")

Use "ruin [command] --help" for more information about a command.

```
